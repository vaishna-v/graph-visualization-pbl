Route Finding Algorithm

About the Project

This interactive graph visualization and pathfinding tool was developed as part of the Data Structures Problem-Based Learning (PBL) course. The application provides an educational platform for understanding graph theory concepts, graph generation algorithms, and pathfinding techniques.

Key Features:

  • Interactive graph visualization with draggable nodes
  • Multiple graph generation models (Random & Sliding Window)
  • Dijkstra's algorithm implementation with battery constraints
  • Real-time path highlighting and visualization
  • Graph persistence with save/load functionality
  • Educational tool for algorithm analysis and understanding

Technology Stack

Frontend

  • HTML5, CSS3, JavaScript (ES6+)
  • D3.js for interactive graph visualization
  • Responsive design with modern CSS Grid/Flexbox

Backend

  • Node.js with Express.js server
  • C++ for high-performance graph algorithms
  • JSON-based communication protocol

Algorithms & Data Structures

  • Graph generation algorithms
  • Dijkstra's shortest path algorithm
  • Min-Heap priority queue implementation
  • Force-directed graph layout algorithms

How to Use

  1. Generate Graphs: Select node count and generation method to create graphs
  2. Interact: Drag nodes to rearrange, zoom and pan to explore large graphs
  3. Find Paths: Set source, destination, battery, and mileage constraints
  4. Visualize: Watch Dijkstra's algorithm find the optimal path in real-time
  5. Save/Load: Persist your graphs for later analysis and comparison

Educational Value

This project serves as an excellent educational tool for understanding:

  • Graph theory fundamentals and representations
  • Algorithm design and complexity analysis
  • Pathfinding algorithms and their practical applications
  • Software architecture and modular design
  • Cross-language integration (C++ & JavaScript)
  • Data visualization principles and techniques
